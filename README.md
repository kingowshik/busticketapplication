## Project Overview
The **NCTX Bus Ticket Application** is a console-based Python program designed to allow users to register, log in, view bus ticket categories, and purchase tickets.

The system uses **CSV files** for data storage and follows **Object-Oriented Programming (OOP)** principles.

---

## Features
- User registration with email validation  
- Secure login using masked password input  
- Viewing available bus ticket categories  
- Viewing tickets within a selected category  
- Purchasing tickets and saving purchase history  
- Help and About section for user guidance  

---

## How to Use
1. Start the application
2. Choose Login or Signup from the home menu
3. After logging in, users can:
   
  - View ticket categories
  - Select available tickets
  - Purchase tickets
  - Access the Help/About section if needed
  - Logout safely when finished
